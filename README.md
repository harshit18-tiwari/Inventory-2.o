# Inventory Management System — Frontend

This is a minimal React + Vite frontend scaffold for the Inventory Management System (IMS) UI. It stores data in `localStorage` for now and demonstrates core frontend workflows: dashboard, product CRUD, suppliers, and transaction log.

Quick start:

```powershell
npm install
npm run dev
```
